Task Manager API

Simple REST API for managing tasks.

Tech stack:
-Python
-FastAPI
-SQLModel
-SQLite

Features:
-Adding a task
-Removing a task
-Updating a task
-List of tasks

How to run:
1.Create virutal environment
2.Install requirements
3.Run uvicorn
4.Open /docs

Notes:
-Database is created on first startup.
